# Probability Calculator

**Purpose:**  
A tool for calculating the probability of winning at least once across multiple groups of events. Useful for analyzing chances in lotteries, raffles, or any sequential events with specified participants and winners.

**Key Features:**  
- **Event Groups** – create any number of groups, each with its own list of events.  
- **Flexible Statuses** – mark events as won, lost, or pending. Probabilities are automatically recalculated based only on relevant events.  
- **Profiles** – save different datasets into separate profiles, export and import them as JSON.  
- **Summary Statistics** – overall win chance, counts of wins/losses/pending, and top‑3 best chances by event name.  
- **Backup & Restore** – create and restore a full copy of all profiles.  
- **Dark / Light Theme** – automatic system theme detection or manual toggle.  
- **Russian and English interface** support.

**Hotkeys:**  
- ↑/↓ arrows – navigate between input fields.  
- `Ctrl + Enter` – quickly add a new event in the current group.

**Getting Started:**  
1. Create or load a profile.  
2. Set the number of groups and configure names and event counts.  
3. Switch to the calculator, enter participants and winners for each event.  
4. Update statuses as events conclude – probabilities refresh instantly.
