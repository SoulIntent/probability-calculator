# Probability Calculator

**Назначение:**  
Инструмент для расчёта вероятности выигрыша хотя бы в одном событии из нескольких групп. Удобен для анализа шансов в лотереях, розыгрышах или любых последовательных событиях с заданными участниками и победителями.

**Основные возможности:**  
- **Группы событий** – создавайте произвольное количество групп, каждая со своим списком событий.  
- **Гибкие статусы** – отмечайте события как выигранные, проигранные или ожидающие. Вероятность автоматически пересчитывается только по актуальным событиям.  
- **Профили** – сохраняйте разные наборы данных в отдельные профили, экспортируйте и импортируйте их в JSON.  
- **Сводная статистика** – общий шанс на победу, количество побед/поражений/ожиданий, топ‑3 лучших шанса по названиям событий.  
- **Резервное копирование** – создание и восстановление полной копии всех профилей.  
- **Тёмная / светлая тема** – автоматическое определение системной темы или ручное переключение.  
- **Поддержка русского и английского языков** интерфейса.

**Горячие клавиши:**  
- Стрелки ↑/↓ – перемещение между полями ввода.  
- `Ctrl + Enter` – быстрое добавление нового события в текущей группе.

**Как начать:**  
1. Создайте или загрузите профиль.  
2. Укажите количество групп и настройте названия, число событий.  
3. Перейдите в калькулятор, заполните участников и победителей для каждого события.  
4. Меняйте статусы по мере прохождения — вероятность обновляется мгновенно.
